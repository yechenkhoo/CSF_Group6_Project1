LSB Steganography Tool

Hide and extract secret data using Least Significant Bit (LSB) replacement techniques in images and audio files.

This tool supports custom binary payloads, user-defined keys, and advanced steganographic analysis features.

Encode Data

Hide secret data inside cover objects (images or audio files) using LSB replacement.

  • Support for BMP, PNG, GIF images
  • Support for WAV, PCM audio
  • Custom payload support
  • User-defined encryption keys
  • Selectable LSB depth (1-8 bits)
Start Encoding
Decode Data

Extract hidden data from stego objects using the correct decryption key.

  • Automatic file type detection
  • Key-based extraction
  • Payload verification
  • Multiple output formats
  • Error detection
Start Decoding
Analyze

Perform steganalysis and visualize changes made to cover objects.

  • Visual difference maps
  • LSB change visualization
  • Statistical analysis
  • Waveform comparison
  • Capacity analysis
Start Analysis
Features Overview
Core Features
  • Multi-format Support: Images (BMP, PNG, GIF) and Audio (WAV, PCM)
  • Custom Payloads: Text, images, executables, PDFs, and custom binary data
  • LSB Selection: Choose from 1-8 LSBs for embedding depth control
  • Drag & Drop Interface: Easy file selection and management
  • Location Selection: Click to choose embedding location in cover objects
Security Features
  • User-Defined Keys: Numeric keys influence LSB selection and permutation
  • Custom Key Format: Flexible metadata structure for enhanced security
  • Capacity Checking: Automatic validation of payload size vs. cover capacity
  • Error Detection: Comprehensive validation and error reporting
  • Visualization: Real-time analysis of changes and differences
Educational Tool

This steganography tool is designed for educational purposes to demonstrate LSB replacement techniques and steganalysis methods. Always use responsibly and in accordance with applicable laws and regulations.
